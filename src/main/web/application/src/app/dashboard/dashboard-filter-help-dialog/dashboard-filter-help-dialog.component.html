Filter on column headers

  • The filter text is applied on columns, only on column header names. If a column has a match, it is displayed.

  • The matching is not case sensitive.

  • Multiple filter texts is not allowed.

Filter on job name, build number, helm version

  • The filter text is applied on rows, both on service name and cells' content. If a row has a match, it is displayed.

  • The matching is not case sensitive.

  • Multiple filter texts can be combined with '||' (logical OR), then the whole expression is matched to the rows.

    Example: log || failed. The full rows that have 'log' or 'failed' in service name or in cell content are displayed.
